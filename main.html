Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: It reads both the HTML markup and the text content of the element.
innerText: It ignores HTML tags. And it also does not include text that is hidden with CSS styles.
textContent: It only deals with the raw text and doesn't account for styles.

Question-3: What is event delegation in the DOM?

It allows us to attach a single event listener for elements that exist now or in the future.

Question-4: What is event bubbling in the DOM?

It starts with an event triggering on the innermost element and then proceeds to trigger the same event on its parent element and then its parent's parent element and so on.
